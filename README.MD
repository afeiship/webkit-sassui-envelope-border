# webkit-sassui-envelope-border
> Envelope border for webkit.

## usage:
```sass
//import:
@import 'bower_components/webkit-sassui-envelope-border/dist/webkit-sassui-envelope-border.scss';

//use this mixin:
@include webkit-sassui-envelope-border(
  $inSize:5px,
  $inBgSize:10%,
  $inRadius:10px,
  $inBdPadding:10px
);
```
